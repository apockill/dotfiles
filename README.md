# Installation

1) Create a copy of default_env_vars.sh, and call it custom_env_vars.sh (name doesn't matter)
2) Run ```python3 dependencies.py```
3) Run ```bash cat helpful/scripts/.bashrc-addon.sh >> ~/.bashrc```
